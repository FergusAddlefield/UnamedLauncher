## Installation

1. Télécharger `UnamedLauncher.exe` depuis le lien ci-dessus
2. Placer l'exécutable à l'emplacement de votre choix
3. Lancer `UnamedLauncher.exe`

> Aucune installation requise. Le launcher se met à jour automatiquement à chaque démarrage.
